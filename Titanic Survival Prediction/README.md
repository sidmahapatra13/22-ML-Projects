# Titanic Survival Prediction

A short machine learning project that explores the Titanic dataset and predicts passenger survival using classic classification models.

## What's Inside

- Exploratory data analysis on passenger demographics and travel details
- Feature engineering from columns like `Cabin`, `Ticket`, and `Name`
- Preprocessing with missing value handling, dummy encoding, and scaling
- Model comparison using Logistic Regression, Decision Tree, KNN, Random Forest, SVC, and a Voting Classifier

## Results

The notebook compares baseline models and tuned performance, with the strongest results reaching about **83.6% accuracy**.

## Files

- `titanic_pred.ipynb` — main notebook
- `titanic-train.csv` — training dataset
- `titanic-test.csv` — test dataset

## Tech Stack

Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.
